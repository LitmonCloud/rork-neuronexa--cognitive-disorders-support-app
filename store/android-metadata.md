   - Try the Wellness tab for breathing exercises
   - Visit Settings to explore accessibility options
   - Check the Coach tab to chat with Nexa

4. **AI Integration:** We use Rork Toolkit SDK for AI features. No external API keys are required for testing.

5. **Offline Support:** The app works fully offline. AI features require internet but have fallback responses.

6. **Medical Disclaimer:** Clearly displayed in Settings and onboarding. This is NOT a medical device.

7. **Privacy:** All data is stored locally. No analytics or tracking in this version.

8. **Accessibility:** Tested with TalkBack. High contrast and large text modes available in Settings.

If you have any questions, please contact support@neuronexa.app.
