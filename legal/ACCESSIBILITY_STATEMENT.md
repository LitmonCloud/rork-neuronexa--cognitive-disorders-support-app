# Accessibility Statement for Nexa

**Effective Date:** October 2, 2025  
**Last Updated:** October 2, 2025

---

## Our Commitment to Accessibility

Nexa is committed to ensuring digital accessibility for people with disabilities, cognitive differences, and neurodivergent individuals. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

**Our mission is to provide cognitive support that is accessible to all users, regardless of their abilities.**

---

## Accessibility Features

### Visual Accessibility

**High Contrast Mode**
- Enhanced color contrast ratios (WCAG AAA compliant)
- Clear visual separation between UI elements
- Reduced visual noise and distractions
- Toggle: Settings → Accessibility → High Contrast

**Large Text Mode**
- 1.2× text scaling across the entire app
- Maintains readability and layout integrity
- Respects system font size preferences
- Toggle: Settings → Accessibility → Large Text

**Reduced Motion**
- Disables animations and transitions
- Reduces visual distractions
- Improves focus for users sensitive to motion
- Toggle: Settings → Accessibility → Reduced Motion

### Cognitive Accessibility

**Cognitive Support Mode**
- Simplified language and instructions
- Step-by-step guidance with visual cues
- Reduced cognitive load
- Clear, consistent navigation
- Toggle: Settings → Accessibility → Cognitive Support

**AI Coach Adaptation**
- Nexa learns your communication preferences
- Adjusts language complexity to your level
- Provides scaffolded support
- Offers multiple explanation styles

**Visual Cues**
- Icons accompany all text labels
- Color is never the only indicator
- Clear visual hierarchy
- Consistent design patterns

### Auditory Accessibility

**Text-to-Speech Support**
- Compatible with iOS VoiceOver
- Compatible with Android TalkBack
- All UI elements properly labeled
- Descriptive button and action labels

**Haptic Feedback**
- Tactile confirmation for actions
- Breathing exercise pacing
- Task completion celebrations
- Can be disabled in Settings

### Motor Accessibility

**Touch Targets**
- Minimum 44×44pt touch targets (WCAG 2.1 compliant)
- Generous spacing between interactive elements
- No precision gestures required
- Single-tap interactions preferred

**Gesture Alternatives**
- All gestures have button alternatives
- No complex multi-touch required
- Swipe actions have tap equivalents

### Neurodiversity Support

**ADHD Support**
- Clear task prioritization
- Visual timers and reminders
- Reduced distractions
- Dopamine-friendly progress tracking

**Autism Support**
- Predictable navigation patterns
- Clear expectations and instructions
- Sensory-friendly design
- Routine and structure support

**Executive Function Support**
- Task breakdown and scaffolding
- Step-by-step guidance
- Memory aids and prompts
- Decision-making support

## Known Limitations

We are aware of the following accessibility limitations and are working to address them:

**Current Limitations:**
1. **Finger Tracing Exercises:** May be challenging for users with fine motor difficulties. We are exploring alternative input methods.
2. **AI-Generated Content:** Occasionally may use complex language. We are improving Nexa's language adaptation.
3. **Breathing Exercise Timing:** Fixed timing may not suit all users. Customizable timing is planned.

**Planned Improvements:**
- Voice input for task creation
- Customizable breathing exercise timing
- More granular text size controls
- Additional language simplification options
- Keyboard shortcuts (web version)
